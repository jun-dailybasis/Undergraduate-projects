<Usage>
——————————————————————————————————————————————————————————————
This application consists of 8 functions. it can help users to handle
PDF files conveniently with them. It provides shortcut key function.
You can use the following functions. 
01. You can print PDF file on the both sides.
02. You can make a list of files in a selected file on the txt, word and excel.
03. You can convert word file to pdf file.
04. When printing a PDF file,
    you can print the file name in the upper right corner and page number in the bottom center.
05. You can change name of files collectively.
06. You can change path of files collectively.
07. You can search word what you want on the pdf file and see the result on the excel
08. You can print PDF files collectively on the both sides.
